Projet


I learned the different types of computer network architectures. Including how to install, administer, protect and manage equipment maintenance. The objective of setting up this architecture is to discover the different services and also to see the resulting security principles such as QOS (Quality Of Services), PSI (IT Security Plan) and High availability.



In this 3-tier network infrastructure we can see two web servers changed to maintain service continuity with the protocol Network load balancing and dissemination of the website (IIS, Web platform) and a database server (mysql Navicat). We see two ADs that allow service continuity with fault tolerance and redundancy. A monitoring server (PRTG) that will alert, a firewall that filter and control incoming and outgoing exchanges and packets between lan, wan, dmz. finally users with whom we can give Group policy object gpo, password strategies
